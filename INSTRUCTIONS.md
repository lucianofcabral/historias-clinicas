### Formatos y Localizaci√≥n (Argentina) üá¶üá∑
- **Fechas:** SIEMPRE formato ISO 8601 `YYYY-MM-DD` (ej: `2025-10-18`)
  - En base de datos: tipo `date` o `datetime`
  - En inputs HTML: `type="date"` (est√°ndar ISO)
  - En displays: mantener `YYYY-MM-DD` para consistencia
  - NO usar formatos DD/MM/YYYY o MM/DD/YYYY
- **N√∫meros:**
  - Separador decimal: coma `,` (ej: `36,5¬∞C`, `1,75 m`)
  - Separador de miles: punto `.` (ej: `1.000`, `10.500`)
  - En c√≥digo Python: usar `locale.setlocale(locale.LC_ALL, 'es_AR.UTF-8')`
  - Para formateo: `f"{numero:,.2f}".replace(",", "X").replace(".", ",").replace("X", ".")`
- **DNI:** Formato argentino sin puntos ni guiones (ej: `12345678`)
  - Validaci√≥n: 7-8 d√≠gitos num√©ricos
  - Almacenar como string para preservar ceros iniciales
- **Tel√©fonos:** Formato flexible argentino
  - Con c√≥digo de √°rea: `(011) 1234-5678` o `011-1234-5678`
  - Celular: `15-1234-5678` o `011-15-1234-5678`
- **Moneda:** Pesos argentinos
  - S√≠mbolo: `$` o `ARS`
  - Formato: `$ 1.234,56`

```python
# app/utils/security.py
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def verify_password(plain_password: str, hashed_password: str) -> bool:
    """Verifica si la contrase√±a es correcta"""
    return pwd_context.verify(plain_password, hashed_password)

def hash_password(password: str) -> str:
    """Genera hash de contrase√±a"""
    return pwd_context.hash(password)
```

```python
# app/state/auth_state.py
import reflex as rx
from app.config import ADMIN_PASSWORD_HASH
from app.utils.security import verify_password

class AuthState(rx.State):
    """Estado de autenticaci√≥n simple"""
    is_authenticated: bool = False
    login_error: str = ""
    
    def login(self, password: str):
        """Intenta hacer login"""
        if verify_password(password, ADMIN_PASSWORD_HASH):
            self.is_authenticated = True
            self.login_error = ""
            return rx.redirect("/dashboard")
        else:
            self.login_error = "Contrase√±a incorrecta"
    
    def logout(self):
        """Cierra sesi√≥n"""
        self.is_authenticated = False
        return rx.redirect("/login")
    
    def check_auth(self):
        """Verifica si est√° autenticado (para proteger p√°ginas)"""
        if not self.is_authenticated:
            return rx.redirect("/login")
```
